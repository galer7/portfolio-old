## ğŸ§ª Tech stack

For this project I've used Next.js and TailwindCSS for the front-end + server-side rendered pages, and Socket.IO for the backend, both types in TypeScript.
The Next.js app is deployed on Vercel, while the WebSocket server is deployed on fly.io, using a simple multi-stage Dockerfile.

## ğŸ¾ First steps: Next.js app

I've started this project with little to no experience in Next.js, React and/or TailwindCSS (pretty much anything frontend really).
This is also the first project I've completed using TypeScript.
